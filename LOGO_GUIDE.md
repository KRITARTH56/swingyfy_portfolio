# SwingyFy Logo Guide

## Logo Variations

### 1. Full Logo (`logo.svg`)
- **Size**: 120x120px
- **Usage**: Main logo for hero sections, marketing materials
- **Features**: Full "S + Flow Orbit" design with gradient colors

### 2. Icon Logo (`logo-icon.svg`)
- **Size**: 40x40px
- **Usage**: Navbar, favicon, app icons, social media profiles
- **Features**: Simplified icon with rounded square background

### 3. Text Logo (`logo-text.svg`)
- **Size**: 200x40px
- **Usage**: Horizontal layouts, email signatures
- **Features**: Text-only version with gradient

### 4. Full Logo with Text (`logo-full.svg`)
- **Size**: 180x50px
- **Usage**: Headers, documentation, presentations
- **Features**: Icon + text combination

### 5. White Logo (`logo-white.svg`)
- **Size**: 120x120px
- **Usage**: Dark backgrounds, print materials
- **Features**: White version for contrast

## Design Concept

The logo represents:
- **S**: The letter "S" for SwingyFy
- **Flow Orbit**: Circular orbits representing workflow automation and smooth HR processes
- **Particles**: Flow elements showing movement and efficiency
- **Gradient**: Pink to purple to blue representing modern, tech-forward brand

## Color Palette

- **Primary Pink**: #f7598f
- **Primary Pink Dark**: #e03568
- **Purple**: #8E44FF
- **Blue**: #4FC3F7

## Usage Guidelines

### Minimum Sizes
- Full logo: 60px minimum
- Icon: 24px minimum
- Text: 16px minimum

### Clear Space
- Maintain padding equal to 20% of logo height around all sides
- Never place text or elements closer than this distance

### Backgrounds
- Use colored logo on dark backgrounds
- Use white logo on very dark backgrounds
- Never use logo on busy or cluttered backgrounds

### Do's ✅
- Use logo at appropriate sizes
- Maintain aspect ratio
- Use correct color variations
- Keep clear space around logo

### Don'ts ❌
- Don't stretch or skew the logo
- Don't change colors arbitrarily
- Don't add effects or filters
- Don't place on low-contrast backgrounds
- Don't rotate the logo

## File Formats

- **SVG**: Primary format for web (scalable, crisp)
- **PNG**: For applications requiring raster format
- **ICO**: For favicons (converted from SVG)

## Implementation

The logo is implemented in:
- Navbar component (icon version)
- Footer component (icon version)
- Favicon (app/icon.svg)
- All marketing materials

## Brand Personality

The logo conveys:
- Modern and tech-forward
- Reliable and professional
- HR-friendly and people-focused
- Premium but approachable
- Automation and intelligence
